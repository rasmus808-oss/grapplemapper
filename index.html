How to use
  • Click a node → turns purple (armed)
  • Click another node → arrow drawn
  • Esc or click armed node → cancel
  • Drag node → move it
  • Double-click node → edit details
  • Double-click canvas → add node
  • Drag canvas → marquee select
  • Del / Backspace → delete selected
  • Click arrow → delete connection
  • Double-click arrow → add/edit label
Edit technique
Click a node to arm it · click another to connect · drag canvas to pan
